# Code of Conduct

We follow the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).
In short: be respectful, be inclusive, assume good faith, and focus
on technical merit. Harassment, personal attacks, and discriminatory
language have no place in this project.

Report unacceptable behavior to the maintainers (see GOVERNANCE.md).
Reports are kept confidential. Action will be proportionate to the
incident.
